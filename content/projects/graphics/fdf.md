## Overview

FdF (fil de fer) is a graphics project from 42 School where the goal is to create a 3D representation of a 2D map. It reads coordinates from a file and renders a 3D isometric wireframe using Bresenham's line algorithm.

## Features

- Custom implementation of Bresenham's line algorithm
- Isometric and parallel projection
- Window management and event handling (keyboard/mouse)
- Color parsing and gradients
