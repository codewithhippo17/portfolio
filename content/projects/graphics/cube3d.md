## Overview

Cube3D is a graphics project where the objective is to create a realistic 3D graphical representation of the inside of a maze from a first-person perspective, using the raycasting principles from the world-famous game Wolfenstein 3D.

## Features

- Smooth raycasting implementation
- Texture mapping for walls
- Collision detection and player movement
- Map parsing and validation
