## Overview

Energy Copilot - OCP (Optimization & Control Platform) is a Next.js-based energy optimization and monitoring platform. It features AI-powered chat capabilities utilizing Ollama and OpenCode to provide intelligent insights into energy consumption and control systems.

## Features

- Real-time energy monitoring dashboard
- AI-driven chat assistant for querying metrics and controlling systems
- Built with modern web technologies: Next.js and TypeScript
- Integration with local LLMs via Ollama for privacy and speed
